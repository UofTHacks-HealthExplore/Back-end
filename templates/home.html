This is the home page. Below contains explanation of various redirects

Testing redirects

/test

Questions

/chatlog - returns a page containing full chat history

/last_question - returns the last question asked by the AI

Responses

/gen_responses - generates 5 possible responses. Run this command before any other response command

/responses/all - After responses are generated, this command allows for all responses to be seen. More information on sending responses available on this page.

NPCs

/send_personalized?q=<query> - when entering stage 3, this is the answer thats inserted into the backend "database" if the user is comfortable with revealing more personal details.

/gen_tips - when entering stage 1, running this command generates a random mental health tip, passing in the chat stored in the "database" in the process. TRY NOT TO SPAM

/gen_articles - when entering stage 2, running this command generates a random mental health article, video, other resource, passing in the chat stored in the "database" in the process. TRY NOT TO SPAM

/gen_personalized - when entering stage 3, running this command generates a random mental health resource thats more detailed given what information the user provided upon entering stage 3. Passes in the "database" stored in the process. TRY NOT TO SPAM

Other

/reset - Resets all data
